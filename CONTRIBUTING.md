# Contributing Guidelines

Thank you for your interest in contributing to MOS3. These are the contribution guidelines.  
Please read through this document before submitting any issues or pull requests.

## Contributing via Pull Requests

To send a pull request, please:

1. Fork the repository.
2. Modify the source code.
3. Ensure local tests pass.
4. Commit the changes to your fork.
5. Send a pull request.

> [!NOTE]
> You can install md-outline.nvim in development mode by specifying `dev = true` in `~/.config/nvim/lua/plugins/md-outline.lua`.
> ```lua
> return {
> 
> dir = "path/to/repository/that/you/cloned/md-outline.nvim",
>   name = "md-outline",
>   dev = true,
> }
> 
> ```
