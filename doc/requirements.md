# 要件
md-outlineはmarkdownファイルの編集体験を向上させるプラグインです。具体的な機能としては、現在のバッファで開いている.mdファイルから見出し行を抽出し目次のアウトラインを作成し、画面右側に表示するものとなります。

# ルール
- 開発言語はluaを使うこと
- 開発初期段階では、一旦モジュール分けのことは意識せずに全てのロジックをlua/md-outline.luaに記述してください。ある程度開発が進んでからモジュール分けのことを考えます。
- 私はneovimのプラグイン開発をするのが初めてです。プラグイン開発におけるベストプラクティスや慣例などは詳しくありません。私がベストプラクティスや慣例から外れた行為をしていたら、指摘してください。
